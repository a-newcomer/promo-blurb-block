== Description ==

A simple static custom Gutenberg block for calls to action on an editor's site, which features an easy-to-use headline, text and link.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/promo-contact-blurb` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
